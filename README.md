# 📘 Claude Code 进阶实用教材

> 面向"已安装 Claude Code，但想系统掌握多样用法"的学习者。

------

## 🎯 这套教材适合谁

- 已完成 Claude Code 安装，想进一步释放其能力的开发者
- 希望建立**系统化 AI 辅助编程工作流**的团队或个人
- 对 Claude Code 的高级特性（子 Agent、Hooks、MCP 等）感兴趣的探索者

## ✨ 本教材特色与亮点

- **前沿同步**：内容深度对齐 2026 年最新官方文档，涵盖最新的持久记忆（`CLAUDE.md`）、标准子 Agent 管理（`/agents`）、生命周期钩子（Hooks）及脚本等新特性。
- **实战导向**：抛弃宽泛的 AI 废话，每一章都提供真实开发场景中的**推荐提示词**与**避坑指南**。
- **安全第一**：严守执行边界，建立起面对删文件、改权限、碰密钥、装依赖等高风险操作时的肌肉记忆。

## 🧱 每章统一结构

每一章都遵循相同的七段式框架，确保学习体验的一致性：

1. **使用场景** — 适合 / 不适合的情境说明
2. **推荐提示词模板** — 可直接复用的高质量 prompt
3. **错误示范 + 分析** — 常见踩坑与原因拆解
4. **正确示范** — 对比展示最佳实践
5. **练习任务** — 动手巩固所学内容
6. **验收标准** — 明确"做对了"的判定依据
7. **官方依据** — 附带访问日期的官方文档引用

## ⚠️ 安全约定

> **以下操作必须先看计划和 diff、确认影响后再放行：**
>
> 删除文件 · 批量重构 · 改权限 · 安装依赖 · 提交代码 · 联网访问 · 处理密钥或凭证

每章开头都会额外提醒该章特有的风险点。
